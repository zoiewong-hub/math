# Sudoku Pro（全能数独软件）

一个纯前端数独应用，支持：

- 用户注册 / 登录（本地离线账号）
- 不同难度新局（简单 / 中等 / 困难）
- 笔记模式（候选数标记）
- 冲突高亮、同组高亮、同数字高亮
- 错误次数限制、提示功能
- 计时、进度、动态得分
- 每个用户独立的历史得分记录
- 每个账号独立保存个性化设置（主题、输入模式、难度、标识开关、笔记模式）
- 多主题配色（经典蓝 / 高对比黑金 / 薄荷绿 / 日落暖橙）
- 两种输入逻辑：先选格再填数、先选数字连填（支持长按数字进入连填模式）
- 错误提示增强：错误后强提示，3 次错误弹出游戏结束窗口

## 启动方式

```bash
python -m http.server 8000
```

浏览器打开：`http://localhost:8000`

> 所有数据保存在浏览器 `localStorage`。
